# Visual and Kinesthetic Accommodation Guide

## What

This guide adapts the lab for visual and hands-on learning.

It is written for ADHD-friendly and Autism-friendly study.

## Why

The learner should not need to translate long abstract instructions before taking action.

Good lab design makes the next step visible.

Example:

Use a blue card for network tasks, a green card for completed tickets, and a red card for safety warnings.

## How

Use visual supports:

- [ ] Color-code lab zones.
- [ ] Put the success condition at the top of each lab.
- [ ] Use diagrams before command examples.
- [ ] Use one-page checklists.
- [ ] Use visible ticket movement.

Use kinesthetic supports:

- [ ] Label real cables.
- [ ] Sort parts into trays.
- [ ] Match cards to ports.
- [ ] Trace the network path with a finger.
- [ ] Move a ticket card after each stage.

Use attention supports:

- [ ] One ticket at a time.
- [ ] One screen or page at a time.
- [ ] Timer visible.
- [ ] Break after each ticket.
- [ ] Stop when mistakes increase.

Use predictability supports:

- [ ] Same start checklist.
- [ ] Same ticket template.
- [ ] Same Done definition.
- [ ] Same evidence folder pattern.

Checklist:

- [ ] I can see the next step.
- [ ] I can touch or move something.
- [ ] I know when I am done.
- [ ] I know how to pause.
