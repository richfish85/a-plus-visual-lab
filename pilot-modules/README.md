# Pilot Modules

## What

This folder holds the simplified lesson format for the A+ Visual Lab.

Each pilot module should read like documentation for a new IT technician.

Every module answers exactly five questions:

```text
What happened?
Why did it happen?
How do I diagnose it?
How do I fix it safely?
How do I document it?
```

## Why

The project has useful material, but the presentation layer became crowded. The pilot modules test a cleaner pattern before the visual design is rebuilt.

The goal is:

```text
one support problem
five clear answers
safe checks only
ticket-ready notes
```
