# BIOS/UEFI

## What

BIOS and UEFI are firmware layers that help start the computer before the operating system loads.

UEFI is the modern replacement for legacy BIOS on most current devices.

## Why

This topic helps the learner understand startup, boot mode, secure boot, and why newer systems usually behave differently from older ones.

## How

- Learn that firmware starts the machine before the OS.
- Learn that UEFI is the current standard on modern systems.
- Learn that some devices can still emulate BIOS behavior for compatibility.
- Connect the boot mode to partition style and security behavior.
