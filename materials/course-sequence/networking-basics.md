# Networking Basics

## What

Networking basics is the set of ideas that explain how devices talk to each other and to the internet.

At this level, the learner should know the roles of switches, routers, access points, and DNS.

## Why

This is the foundation for the networking part of A+.

If the learner understands the role of each device, later troubleshooting steps stop feeling random.

## How

- A switch connects devices inside the local network.
- A router connects networks and usually links the local network to the internet.
- An access point gives wireless devices a way onto the network.
- DNS turns human-friendly names into addresses computers can use.

Useful framing:

- Switch = local device traffic.
- Router = between networks.
- Access point = wireless entry.
- DNS = name lookup.
