# System Memory

## What

System memory is RAM.

It is the short-term workspace the computer uses while programs are running.

## Why

RAM is one of the easiest parts to identify in the lab, and it has a useful mental model: if the device is working slowly, memory is one of the things a learner should think about, but not blindly replace.

## How

- Describe RAM as working memory.
- Show where RAM usually sits on the board.
- Teach safe handling by the edges.
- Connect the part to performance symptoms and upgrade choices.
