# Core 2 Map: 220-1202

## What

Core 2 is the operating system, security, software troubleshooting, and process side of entry-level support.

## Why

A new technician needs to support users without damaging data, weakening security, or closing tickets without evidence.

## How To Use This Map

Use each domain as a source pool for short field-manual scenarios.

| Domain | Weight | Technician Translation | Portfolio Evidence |
| --- | ---: | --- | --- |
| Operating Systems | 28% | Install, configure, update, and support Windows, macOS, Linux, and mobile OS basics. | OS settings screenshot, command note, repair checklist |
| Security | 28% | Protect accounts, devices, data, networks, and users from common security risks. | Phishing response note, hardening checklist, access review |
| Software Troubleshooting | 23% | Diagnose crashes, slow systems, failed updates, browser issues, malware symptoms, and mobile app faults. | Before/after checks, error capture, safe repair note |
| Operational Procedures | 21% | Document, communicate, escalate, back up, handle change, and work safely. | Ticket notes, escalation handoff, change record |
