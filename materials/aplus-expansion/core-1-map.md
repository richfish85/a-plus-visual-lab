# Core 1 Map: 220-1201

## What

Core 1 is the physical and connection side of entry-level support: devices, networks, hardware, cloud/virtualization basics, and hardware/network troubleshooting.

## Why

A new technician needs to identify parts, trace connections, test simple paths, and avoid unsafe hardware changes.

## How To Use This Map

Use each domain as a source pool for short field-manual scenarios.

| Domain | Weight | Technician Translation | Portfolio Evidence |
| --- | ---: | --- | --- |
| Mobile Devices | 13% | Support laptops, tablets, phones, docks, mobile displays, ports, batteries, and sync issues. | Laptop/mobile intake checklist, dock troubleshooting note |
| Networking | 23% | Understand IP addressing, DNS, DHCP, ports, wireless, cabling, and small-network symptoms. | Network path diagram, command output, ticket note |
| Hardware | 25% | Identify, install, and safely handle PC parts, storage, printers, power, cooling, and peripherals. | Labeled hardware diagram, safe handling checklist |
| Virtualization and Cloud Computing | 11% | Explain VMs, resources, snapshots, basic cloud service models, and hosted services. | VM resource map, snapshot/change note |
| Hardware and Network Troubleshooting | 28% | Use structured troubleshooting for power, display, storage, printer, wireless, and network faults. | Completed scenario module with diagnosis and resolution |
